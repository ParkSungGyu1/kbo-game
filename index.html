📋 게임 방법

  • 6점에서 시작하여 힌트를 볼 때마다 1점씩 차감
  • 첫 번째 힌트인 생년월일은 자동으로 표시됩니다
  • 추가 힌트: 지명순위 → 포지션 → 경력 → 등번호 → 사진
  • 선수 이름을 정확히 입력하면 정답!
  • 모든 힌트를 다 봐도 못 맞히면 게임 종료
